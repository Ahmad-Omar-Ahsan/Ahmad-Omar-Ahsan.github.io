Vision Transformer
An unofficial TensorFlow implementation of Vision Transformer. Vision transformer introduces attention mechanism to classify images. Attention mechanism has been heavily used in NLP with the introduction of Transformer. In ViT images are divided into equal patches and passed through a dense layer to generate embeddings. It passes through attention blocks to generate encoder embeddings after which a classification head classifies the image

ResNet
An unofficial PyTorch implementation of ResNet. The ResNet model introduces skip connection which adds the input with the result of the block of layers. This skip connections allows deeper models to give better performance. Without skip connections models wouldn’t perform better than their shallower counterpart.
Ahmad Omar Ahsan
Code
EfficientNet
An unofficial PyTorch implementation of EfficientNet. This model introduces compound scaling, a methodology to change the depth, width and resolution of the model in a sustainable manner. Compound scaling enables the model to give higher accuracy compared to scaling a single component. This model also use inverted residual blocks from MobileNetV2 and stochastic depth to improve the performance of the model.
Ahmad Omar Ahsan
Code
Keyword Transformer
An unofficial open source implementation of the keyword transformer. Inspired from Vision Transformer, KWT utilises patches from MFCCs to capture semantic information. The model uses a prenorm transformer encoder block which contains self attention sublayer followed by a classfication head to classify the keywords from Google Speech V2 dataset.
